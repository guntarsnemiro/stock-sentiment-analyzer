# Stock Sentiment Analyzer

A web app that analyzes daily stock market movements and suggests 10 potential buy or short opportunities based on price momentum and reversal patterns.

## Team Members
Guntars Nemiro

## API Used
**Alpha Vantage**  
Documentation: [https://www.alphavantage.co/documentation/](https://www.alphavantage.co/documentation/)

## Project Goals
- Display top 10 stocks with strongest daily price movements.
- Suggest potential buy (oversold) or short (overbought) opportunities.
- Visualize simple trend charts for selected stocks.
