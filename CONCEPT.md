# Stock Sentiment Analyzer

## Team Members
Guntars Nemiro

## Project Description
Web lietotne, kas analizē dienas akciju cenu izmaiņas un iesaka 10 akcijas, kuras potenciāli pirkt vai šortot, balstoties uz pēdējo dienu cenu svārstībām.

## Problem Statement
Investori bieži meklē īstermiņa cenu svārstību iespējas, taču manuāli analizēt desmitiem grafiku aizņem daudz laika. Šī lietotne palīdz ātri redzēt akcijas ar straujām cenu izmaiņām un piedāvā iespējamos “reversal” kandidātus.

## API Selection
**API Name:** Alpha Vantage  
**API Documentation:** [https://www.alphavantage.co/documentation/](https://www.alphavantage.co/documentation/)  
**Why this API?**  
Alpha Vantage piedāvā pilnīgi bezmaksas piekļuvi dienas akciju cenām un ļauj viegli analizēt cenu dinamiku un modeļus.
